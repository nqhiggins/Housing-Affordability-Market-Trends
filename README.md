## Description

This project analyzed housing affordability across 99 Iowa counties using income, rent, and home value data from 2019 to 2022. The goal is to understand how income relates to housing affordability, which counties have seen the biggest cost increases, and whether housing prices and rents are rising faster than incomes in certain areas.

## How to Run Program

Download the following csv files: "Iowa_Counties_FIPS.csv" & "Iowa_Median_Household_Income_in_the_Past_12_Months_(ACS_5-Year_Estimates)_20251113.csv". Then open the Juypter Notebook file and hit run.
